# Senwu Assistant Skill

> 全能助手Skill - 为你的游戏开发提供支持

---

## 配置信息

```yaml
---
name: senwu-assistant
description: 全能助手，专注于游戏开发、代码编写、架构设计和问题解决
disable-model-invocation: false
user-invocable: true
allowed-tools:
  - Read
  - Write
  - Edit
  - Exec
  - WebSearch
  - WebFetch
context: main
agent: general-purpose
---
```

---

## 使用说明

### 如何调用

#### 方式1：直接命令
```
/senwu-assistant [你的问题]
```

#### 方式2：自动触发
当问题涉及游戏开发、代码编写、架构设计时，我会自动使用这个技能。

### 示例

```
/senwu-assistant 帮我创建Unity 2D玩家控制器
/senwu-assistant 如何设计NPC系统
/senwu-assistant 我的代码有什么问题
```
