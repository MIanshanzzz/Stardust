## 🚀 立即开始

### **第一步：安装 Templater 插件**
```
1. 打开 Obsidian → 设置 → 社区插件
2. 搜索 "Templater" → 安装 → 启用
3. 设置 → 外观 → 标签栏 → 添加模板路径
   → C:\Users\Administrator\AppData\Roaming\Obsidian\Vault\.obsidian\templates
```

### **第二步：创建人物卡**
```
Ctrl + ; → 选择 "character-template.md"
在标题处输入 "队长"
点击保存
```

### **第三步：创建世界观**
```
Ctrl + ; → 选择 "world-template.md"
在标题处输入 "神庙遗迹"
点击保存
```

## 💡 推荐写作流程

### **方法1：大纲优先**
```
1. 写章节大纲
2. 写世界观设定
3. 写人物小传
4. 写场景描述
5. 撰写正文
```

### **方法2：场景驱动**
```
1. 选择一个关键场景
2. 用场景模板细化
3. 扩展到整章
4. 完善大纲
```
