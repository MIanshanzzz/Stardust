# 星尘的写作训练计划 v2

## 训练目标
成为成熟耽美写手，帮老大写《Alsa常》

## 目标小说
- **名称**: Alsa常
- **类型**: 悬疑 / 恐怖 / 浪漫
- **核心**: 神明、异化、隐藏的爱
- **设定**: 哀牢山地下遗迹 + 神明守护部落尸体 + 科考队探索
- **人物**：林队（隐藏的感情）+ 沈渊 + 神明

## 学习材料
1. 《21岁》- priest（日常与情感）
2. 《天官赐福》- priest（节奏与氛围）
3. 《在诡异世界变成蜘蛛精》- 岐山娘（动作与节奏）

## Priest风格核心
- 现实感与奇幻结合
- 主角有缺陷但真实，成长线清晰
- 情感含蓄深沉，用细节传递
- 对话自然真实，有性格
- 日常线与剧情线平衡
- 安静中暗藏张力

## 岐山娘风格核心
- 设定新颖（蜘蛛精+诡异世界）
- 幽默元素（内心吐槽、反差萌）
- 动作戏节奏快，画面感强
- 爽点密集
- 第一人称内心独白丰富
- 世界观交代清晰自然

## Alsa常写作方案
### 结构
- 开篇：科考队进入哀牢山地下遗迹（真实感+紧张）
- 中段：触发机关→神明尾随→队长牺牲→主角异化
- 高潮：神明与主角互动→情感爆发→异化完成
- 结尾：开放式结局，留悬念

### 重点
1. 神明不直接出现，用氛围、影子、声音暗示
2. 主角异化：生理+心理双重变化
3. 队长牺牲：惨烈但非重点，重点是冲击主角
4. 感情线：暗线，藏在动作和细节中
5. 参考Priest的日常描写，岐山娘的节奏，融合成"安静中危险，恐怖中浪漫"

### 风格融合
- Priest：真实感、日常、情感细节
- 岐山娘：节奏、爽点、内心独白
- 创造独特氛围：古老遗迹+现代科考+远古神明+隐藏的爱
