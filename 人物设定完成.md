## 🚀 如何使用

### **第一步：安装 Templater 插件**
```
Obsidian → 设置 → 社区插件 → 搜索 "Templater" → 安装 → 启用
```

### **第二步：打开人物卡模板**
```
Ctrl + P → 选择 "character-template.md"
```

### **第三步：为每个角色创建详细的人物卡**
```
选择模板 → 输入姓名 → 选择角色类型 → 填写详细信息
```
