# 📖 如何在Obsidian中查看人物卡

## 方法1：直接打开文件 ⭐ 推荐

1. **打开Obsidian**
2. 在左侧文件浏览器中找到：
   ```
   Vault
   └── 人物-沈渊.md
   └── 人物-林序.md
   └── 人物-赵金山.md
   └── 人物-苏晚.md
   └── 人物-赵军.md
   └── 人物-陈静.md
   └── 人物-王锐.md
   └── 人物-李哲.md
   └── 人物-刘建国.md
   └── 神明-星空史莱姆.md
   ```

3. **点击任意文件**即可查看

## 方法2：通过模板创建

1. **打开Obsidian**
2. 按 `Ctrl + P`（或 `Ctrl + Shift + P`）
3. 输入 "创建新笔记"
4. 在模板菜单中选择 `character-template.md`
5. 输入姓名，点击保存
