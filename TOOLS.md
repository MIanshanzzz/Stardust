# TOOLS.md - Local Notes

Skills define _how_ tools work. This file is for _your_ specifics — the stuff that's unique to your setup.

## What Goes Here

Things like:

- Camera names and locations
- SSH hosts and aliases
- Preferred voices for TTS
- Speaker/room names
- Device nicknames
- Anything environment-specific

## Apps & Tools

### Obsidian
- **路径**: `C:\Users\Administrator\AppData\Roaming\Obsidian`
- **快捷启动**: 运行 `Obsidian` 命令打开
- **版本**: 1.11.5 (最新)
- **笔记库**: 默认在 `Obsidian\Vault`

### Blogwatcher
- **路径**: `C:\Users\Administrator\.openclaw\workspace\blogwatcher-main\blogwatcher.exe`
- **功能**: RSS/Atom博客监控

### OpenAI Whisper
- **Python包**: 已安装 `openai-whisper` 20250625
- **用途**: AI语音转文字
- **模型**: base (默认)

### Python
- **版本**: 3.14.2
- **包管理**: pip
